# Kenya Tech Awards F.E

## How to use

### 1\. Activate the env

```bash
source bin/activate
```

### 2\. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3\. Run Migrations

```bash
./manage.py migrate
```

### 4\. Run Server

```bash
./manage.py runserver
```


>Server Runs on port 8000
