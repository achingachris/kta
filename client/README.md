# Kenya Tech Awards F.E

## How to use


### 1\. Clone this Repository
>For safety, use NodeJS v16.0.0

```bash
git clone git@github.com:achingachris/kta.git
```

### 2\. Navigate to the directory

```
cd kta
```

### 3\. Install dependencies

```
yarn install
# or
npm install
```

### 5\. Run Dev Server

```
yarn dev
# or
npm run dev
```
